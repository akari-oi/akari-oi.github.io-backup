    四边形不等式

    四边形不等式¶

    w(i,j)是定义在整数域的二元函数。
    若对于在定义域范围内的a,b,c,d,a \le b \le c \le d,都有w(a,d) + w(b,c) \ge w(a,c) + w(b,d),则称函数w满足四边形不等式。
    四边形不等式还有另外一种定义方式:若对于在定义域范围内的a,b,a < b,都有w(a,b+1) + w(a+1,b) \ge w(a,b) + w(a+1,b+1),则称函数w满足四边形不等式。
    这两种定义方式完全等价。

    一维线性DP的四边形不等式优化¶

    对于形如

    f_i = min_{0 \le j < i}\{f_j + w(j,i)\}

    的DP转移方程,未经优化转移复杂度为O(N^2)
    若w满足四边形不等式,我们可以优化到O(N\lg N)

    事实上,转移的决策具有单调性。
    设p_i为f_i的最优决策之一(可以看做在最优的前提下p_i尽量小),即f_{p_i}+w(p_i,i)=min_{0 \le j < i}\{f_j + w(j,i)\}。
    那么对于i' \ge i,总有p_{i'} \ge p_i。

    证明:

    然后我们就可以用分治将算法优化至O(N\lg N)级别。
    似乎还有一种二分的方法,实现较为麻烦,感兴趣的同学可以自行查阅资料。

    二维区间DP的四边形不等式优化¶

    对于形如

    f_{i,j} = min_{i \le k < j}\{f_{i,k} + f_{k+1,j} + w(i,j)\}

    的DP转移方程,未经优化转移复杂度为O(N^3)
    若f(注意了是f)满足四边形不等式,我们可以优化到O(N^2)

    转移的决策也具有单调性。 设p_{i,j}为f_{i,j}的最优决策之一,即f_{i,p_{i,j}} + f_{p_{i,j}+1,j} + w(i,j) = min_{i \le k < j}\{f_{i,k} + f_{k+1,j} + w(i,j)\}
    那么有p_{i,j-1} \le p_{i,j} \le p_{i+1,j}

    证明:

    然后我们就可以较快地完成转移。

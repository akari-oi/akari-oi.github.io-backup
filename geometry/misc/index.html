    Pick定理¶

    简介¶

    对于一个所有点都是格点（即横、纵坐标均为正数）的简单多边形（简单多边形可以理解为边不相交的多边形，当然，不包括顶点），设i=内部格点数(注意：不包括边上！)，b=边上格点数，S=多边形面积。
    那么满足

    S=i+\frac b2 - 1

    对于非简单的多边形 {\displaystyle P} ，皮克定理 {\displaystyle A=i+{\frac {b}{2}}-\chi (P)} ，其中 {\displaystyle \chi (P)} 表示 {\displaystyle P} 的欧拉特征数（不知道是啥）。
    高维推广：Ehrhart 多项式 （不知道是啥）

    while( y ) x %= y, x^=y^=x^=y;
    return x;
}

signed main(){
    while( ~scanf( "%d%d%d%d%d%d", &ax, &ay, &bx, &by, &cx, &cy ) ){
        if ( !ax && !ay && !bx && !by && !cx && !cy ) return 0;
        bx -= ax, by -= ay, cx -= ax, cy -= ay,
        printf( "%d\n", ( Abs(bx * cy - by * cx) + 2 - ( gcd( Abs(bx), Abs(by) ) + gcd( Abs(cx), Abs(cy) ) + gcd( Abs(bx - cx), Abs(by - cy ) ) ) ) >> 1 );
    }
    return 0;
}

    POJ1265 Area¶

    题意：有一个机器人初始点为(0,0)，给出它每次行走增加的横坐标与纵坐标，保证最终回到起点(0,0)，求出走过路径构成多边形的内部格点数、边上格点数、面积。

    就是模板题啊。
    比较难求的是多边形内部格点数，面积比较好求。
    有一个有意思的结论：任意一个多边形的面积等于按顺序求相邻两个点与原点组成的向量的叉积之和（这个也可以通过顺时针定积分求得）。（然而我并不知道为什么(*/ω＼*) 边上的格点数可以通过gcd求出。
    最终使用Pick定理求出内部格点数就OK了。

    #include<cstdio>
#include<iostream>
#define rgt register
#define getchar() ( p1 == p2 && ( p2 = bf + fread( bf, 1, 1 << 21, stdin ), p1 = bf ) == p2 ? EOF : *p1++ )
using namespace std;

char bf[1 << 21], *p1, *p2;
template<typename T>
inline void read( rgt T &x ){ x = 0; rgt char t, flg(0);
    for ( t = getchar(); !isdigit(t); t = getchar() ) flg = t == '-';
    for ( ; isdigit(t); t = getchar() ) x = x * 10 + ( t & 15 ); x = flg ? -x : x;
}


int T, N, lx, ly, x, y, dx, dy, ans1, ans2;
// ans1表示面积 * 2
// ans2表示边上的格点数
// 内部格点数 = ( ans1 - ans2 + 2 ) / 2
inline int gcd( rgt int x, rgt int y ){
    while( y ) x %= y, x^=y^=x^=y;
    return x;
}
inline int Abs( rgt int x ){ return x < 0 ? -x : x; }

signed main(){
    read(T);
    for ( int Ti = 1; Ti <= T; ++Ti ){
        read(N), lx = ly = ans1 = ans2 = 0;
        for ( int i = 1; i <= N; ++i )
            read(x), read(y), ans2 += gcd( Abs(x), Abs(y) ),
            x += lx, y += ly, ans1 += x * ly - y * lx, lx = x, ly = y;
        ans1 = Abs(ans1),
        printf( "Scenario #%d:\n%d %d %.1lf\n\n", Ti, ( ans1 - ans2 + 2 ) >> 1, ans2, ans1 * 0.5 );
    }
    return 0;
}

    欧拉定理¶

    对于任意简单多面体，设V=顶点个数，F=面数，E=边数。
    那么满足

    V+F-E=2

    据说它与Pick定理等价(但是不知道为啥)
    它适用于所有多边形啥维度都行*

    证明¶

    如果该多面体有一个面不为三角形，就将其分割为多个三角形；每分一个，顶点个数不变，边增加一条，点增加一条，V+F-E=2保持不变。
    如果该多面体每个面都是三角形，先取出一个三角形，很明显，边、顶点都不变，面减少一个，此时V+F-E=1。
    选取一个至少有一条边不与其他三角形共用的三角形，将它取出，这样有两种情况，一种是有两条边与其他三角形共用，另外一种是有一条边与其他三角形共用。
    前一种情况，面、边减少一个，满足V+F-E=1不变。
    后一种情况，面、顶点减少一个，边减少两个，满足V+F-E=1不变。
    最后只剩下一个三角形，满足V+F-E=3+1-3=1。
    QED.

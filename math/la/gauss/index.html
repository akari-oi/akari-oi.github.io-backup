    高斯消元¶

    引入¶

    给你一个N元一次方程组，怎么求出它的解？ 就像这样子qwq

    \left\{\begin{aligned} x+4y-10z&=-21 \notag \\ 4x+y+z&=9 \notag \\ 3x+y+4z&=17 \notag \end{aligned}\right.

    动笔算一下你就知道解为

    \left\{\begin{aligned} x&=1 \notag \\ y&=2 \notag \\ z&=3 \notag \end{aligned}\right.

    回想你是怎么算出这个解的? 是不是两个方程相加、相减，不断减少方程中未知数的个数，最后只剩一个而得到解？ 其实这就是高斯消元的主要思想啦qwq

    正题¶

    高斯消元就是模拟人手算的过程。 对于方程组

    \left\{\begin{aligned} a_{1,1}x_1+a_{1,2}x_2+a_{1,3}x_3...&=a_{1,n+1} &(1) \\ a_{2,1}x_1+a_{2,2}x_2+a_{2,3}x_3...&=a_{2,n+1} &(2)\\ ...&= ... &(..) \\ a_{n,1}x_1+a_{n,2}x_2+a_{n,3}x_3...&=a_{n,n+1}&(n) \end{aligned}\right.

    我们用(1)方程消去(2)~(n)中所有的x_1,用(2)方程消去(3)~(n)中所有x_2.... 最后再反过来代回去，就得到了所有解。

    for ( rgt int i = 1, r, N1(N+1), i1; i <= N; ++i ){
    i1 = ( r = i ) + 1;
    for ( rgt int j = i1; j <= N; ++j ) if ( Fabs(mat[j][i]) > Fabs(mat[r][i]) ) r = j; //避免出现0
    if ( r != i ) for ( rgt int j = i; j <= N1; ++j ) swap( mat[i][j], mat[r][j] );
    rgt double t(1. / mat[i][i]); mat[i][i] = 1;
    for ( rgt int j = i1; j <= N1; ++j ) mat[i][j] *= t;
    for ( rgt int j = i1; j <= N; ++j ){
        t = mat[j][i], mat[j][i] = 0;
        for ( rgt int k = i1; k <= N1; ++k )
            mat[j][k] -= t * mat[i][k];
    }
}
